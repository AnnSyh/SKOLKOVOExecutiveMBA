#Верстка 1-ой стр на Tailwind CLI

## Установка и запуск
- git clone https://github.com/AnnSyh/SKOLKOVOExecutiveMBA.git - стягиваем проект
- cd SKOLKOVOExecutiveMBA  - переходим в папку SKOLKOVOExecutiveMBA/
- npm i  - установка зависимостей
- npm run start - запуск проекта
- открываем index.html через 'Open with Live Server'

## Используемые технологии:
- Tailwind CLI

## О проекте:
- проект содержит 1 стр (index.html)
